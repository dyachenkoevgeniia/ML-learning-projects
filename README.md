# Learning Projects Portfolio

This repository contains a collection of learning and pet projects focused on data analysis, machine learning, and working with real-world datasets.  
The projects were created for practice, experimentation, and skill development in Data Science and related areas.

## Projects

### Machine Learning / Data Science

- **[Clustering Music Genres](Clustering%20Music%20Genres)**  
  Unsupervised learning project exploring clustering techniques for grouping music tracks based on audio features.

- **[Credit Card Clustering](Credit%20Card%20Clustering)**  
  Customer segmentation using clustering algorithms to identify patterns in credit card usage behavior.

- **[Credit Score Classification](Credit%20Score%20Classification)**  
  Classification models for predicting customer credit scores using financial and behavioral features.

- **[House Rent Prediction](House%20Rent%20Prediction)**  
  Regression models for estimating house rental prices based on property attributes.

- **[Loan Approval Prediction](Loan%20Approval%20Prediction)**  
  Predictive modeling for loan approval decisions using supervised learning techniques.

- **[Music Popularity Prediction](Music%20Popularity%20Prediction)**  
  Analysis and prediction of music track popularity using statistical and machine learning methods.

- **[Real Estate Price Prediction](Real%20Estate%20Price%20Prediction)**  
  Regression-based project for estimating real estate prices from structured property data.

### Data Collection / Data Engineering

- **[Data Collection with an API](Data%20Collection%20with%20an%20API)**  
  Project demonstrating how to retrieve, process, and store data from an external API.

## Key Topics Covered

Across the projects, the following concepts and techniques are explored:

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering
- Supervised learning (classification, regression)
- Unsupervised learning (clustering)
- Model evaluation and validation
- Working with external data sources (APIs)
